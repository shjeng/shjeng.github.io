## 🛠️ 사용 기술

### Backend

- **Language & Runtime**: Java 21
- **Framework**: Spring Boot 2.7.18, Spring MVC, EGovFrame, Mybatis
- **View Engine**: JSP + JSTL
- **Logging**: log4jdbc-log4j2 + Log4j2

### Frontend

- JavaScript, jQuery

### Database

- MariaDB
